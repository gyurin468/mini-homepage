# Mini Homepage – Winter Cafe Guide

한양대학교 ERICA 근처 분위기 좋은 카페 3곳을 소개하는 미니 홈페이지입니다.  
눈 내리는 겨울 분위기를 테마로 제작하였으며, 각 카페 이미지를 클릭하면 해당 카페의 상세 페이지로 이동합니다.

---

## 📌 Preview
- 상단: 눈송이 애니메이션
- 본문: 카페 소개 카드 3개 (프로스콘스 EAST점 / 버니문 / 더스크커피랩)
- 하단: 랜덤 추천 노래 기능

## ✨ Features
### ✔ 눈송이 애니메이션  
CSS animation을 활용하여 상단에 눈이 내리는 효과 구현.

### ✔ 카페 3곳 소개  
카페 이미지 + 간단한 설명  
이미지 클릭 시 해당 카페의 상세 페이지로 이동.

### ✔ 랜덤 추천 노래  
홈페이지 접속 시 무작위로 배경 음악을 추천해주는 기능.
